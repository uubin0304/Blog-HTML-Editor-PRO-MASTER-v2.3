# 🎨 Blog HTML Editor Pro Master v2.3

> **"코딩은 에디터가 합니다. 당신은 글쓰기에만 집중하세요."**
> 블로그 상위 노출(SEO)과 수익화(쿠팡 파트너스)에 최적화된 **단 하나의 HTML 파일** 에디터입니다.

---

## 📖 프로젝트 소개

이 프로젝트는 워드프레스, 티스토리, 블로그스팟 등 HTML을 지원하는 모든 블로그 플랫폼에서 **전문가 수준의 포스팅**을 할 수 있도록 돕는 웹 도구입니다. 서버 설치나 복잡한 설정 없이, 다운로드 받은 파일 하나만 실행하면 어디서든 바로 사용할 수 있습니다.

특히 **워드프레스 구텐베르크** 에디터의 복잡함에 지친 분들이나, **쿠팡 파트너스** 활동을 위한 버튼/공정위 문구 삽입이 잦은 분들에게 최적화되어 있습니다.

## ✨ 핵심 기능 (Key Features)

### 1. 🚀 수익화 및 생산성 도구
- **원클릭 테마 적용**: 20가지 프리미엄 디자인(색상, 폰트, 간격)을 버튼 하나로 코드에 입혀줍니다. (인라인 스타일 방식)
- **제휴 마케팅 최적화**: 쿠팡 파트너스 등 수익형 링크를 위한 **'CTA(클릭 유도) 버튼'** 생성기와 **'공정위 문구'** 자동 삽입 기능을 제공합니다.
- **앱 다운로드 박스**: 구글 플레이/앱스토어 링크를 깔끔한 박스 형태로 만들어줍니다.

### 2. 🔍 SEO (검색 엔진 최적화)
- **실시간 SEO 분석기**: 글자 수, H2 태그 개수, 이미지 ALT 태그 누락 여부를 실시간으로 하단 바에서 체크합니다.
- **자동 목차 생성 (TOC)**: 본문의 소제목(H2, H3)을 감지하여 하이퍼링크가 걸린 목차를 자동으로 생성합니다.

### 3. 🛠️ 코드 관리 및 편의성
- **코드 클리너 (Code Cleaner)**: 다른 곳에서 복사해 온 글의 지저분한 스타일(폰트, 배경색 등)을 클릭 한 번으로 제거합니다.
- **HTML ↔ Markdown 변환**: 마크다운 문법이 편한 분들을 위한 양방향 변환기를 내장했습니다.
- **완벽한 프리뷰**: 에디터가 다크 모드여도, 미리보기 화면은 실제 블로그와 똑같이 **흰색 배경**을 유지합니다.
- **자동 저장**: 브라우저를 실수로 닫아도 작성 중이던 내용은 안전하게 저장됩니다.

## 🚀 시작하기 (Quick Start)

이 프로그램은 **설치가 필요 없습니다.**

1. 상단의 `index.html` 파일을 다운로드합니다.
2. 다운로드한 파일을 **크롬(Chrome), 엣지(Edge), 웨일(Whale)** 등 웹 브라우저로 엽니다.
3. 바로 글쓰기를 시작하세요!

---

## 🎮 사용 방법 (How to use)

### Step 1. 글 작성하기
왼쪽 에디터 창에 자유롭게 글을 작성하세요.
- 제목은 `<h2>제목</h2>` 형태로 쓰거나, 글자를 드래그한 뒤 단축키를 활용할 수 있습니다.
- 외부에서 쓴 글을 복사해서 붙여넣어도 됩니다. (지저분하다면 **[🧹 코드청소]** 클릭!)

### Step 2. 디자인 입히기
1. 상단 툴바의 **[🌈 테마선택]**을 눌러 마음에 드는 색상을 고릅니다.
2. **중요:** 반드시 **[🎨 코드에 적용]** 버튼을 눌러주세요. 이 과정을 거쳐야 블로그에 붙여넣었을 때 디자인이 유지됩니다.
3. 필요하다면 **[✨ 버튼]**이나 **[🖼️ 이미지]**를 추가하여 글을 풍성하게 만듭니다.

### Step 3. 발행하기
1. **[📋 전체복사]** 버튼을 클릭합니다.
2. 내 블로그 글쓰기 화면에서 **'HTML 모드'**로 변경합니다.
3. 붙여넣기(Ctrl+V) 후 발행하면 끝!

## ⌨️ 단축키 및 팁

- **Ctrl + Z**: 실행 취소 (실수로 내용을 지웠을 때 유용합니다)
- **Ctrl + Y**: 다시 실행
- **프리뷰 클릭**: 오른쪽 미리보기 화면에서 특정 글자를 클릭하면, 왼쪽 에디터의 해당 코드로 즉시 이동합니다.

---

## ⚠️ 주의사항

- **이미지 업로드**: 이 에디터는 이미지를 서버에 저장하지 않습니다. 이미지는 블로그에 먼저 업로드하여 URL을 따오거나, 이미지 무료 호스팅 사이트의 주소를 사용해야 합니다.
- **워드프레스 사용자**: 이미지 삽입 시 '워드프레스 호환 모드'를 체크하면 구텐베르크 블록과 완벽하게 호환됩니다.
